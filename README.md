# FuzzForge
# Intelligent Fuzzing & AI-Powered Crash Triage System

An end-to-end automated fuzzing pipeline that detects software crashes, analyzes them using an LLM, and provides actionable insights for debugging.

---

## 🧠 Overview

This project integrates:

- 🧪 **Fuzzer Engine (M1)** — Generates inputs & detects crashes  
- ⚙️ **Backend System (M2)** — Processes events & stores data  
- 🤖 **LLM Triage (M3)** — Analyzes crashes and suggests fixes  
- 🎯 **Frontend (M4 - upcoming)** — Visualizes results (optional phase)

---

## 🔥 Key Features

- Real-time fuzzing and crash detection  
- Structured event pipeline (JSON-based communication)  
- Automatic crash deduplication  
- AI-powered root cause analysis  
- Mutation chain tracking  
- Session-based fuzzing metrics  
- Scalable backend architecture
